Lo descrito a continuación le ayudará a hacer un mejor uso de la aplicación y facilitará la evaluación de los requerimientos:

1). El sistema está sin productos. Cargue en el sistema Inventario el archivo "EjemploArchInventario.csv".
2). Para usar las funciones de la ventana del sistema Inventario es indispensable elegir un producto primero. Tiene tres opciones "1234", "1444" y "4567".
3). Para probar la función de eliminar vencidos recomendamos usar el producto "1444", pues este estará vencido al momento de que haga la revisión (verifique que la fecha de su PC está bien).
4). Los productos ya vienen con imágenes predeterminadas (las podrá ver en POS). Si desea probar la función de cambio de imagen en la ventana Inventario, sugerimos que use "prueba.png".
5). Para usar las funciones de la ventana del sistema POS es indispensable elegir un cliente primero.
